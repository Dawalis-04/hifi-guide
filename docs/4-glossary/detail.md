# 解析力：480p vs 4K

同一部电影，480p 和 4K 的区别：480p 能看清主角的脸。4K 能看清主角脸上的毛孔、背景里路人的表情、窗户玻璃上反射的街景。这些东西一直都在，只是 480p 让你看不清。

**解析力就是声音的"分辨率"。**

解析好的耳机：录音棚里有人挪凳子、歌手唱完叹气、吉他手手指滑过琴弦的摩擦声——全给你。解析差的耳机：这些细节还在，但像隔了一层纱，你听不太清。

**但解析力强不一定是好事。** 当你在 4K 屏幕上看到你最爱的演员的每一个毛孔时——有些人觉得"哇好真实"，有些人觉得"我宁愿不看这么清楚"。声音也一样。解析高的耳机忠实但不一定好听。解析低一点的耳机可能更"顺滑"更耐听。
