# 瞬态：声音起得快不快、收得利不利落

> **先记一句人话：** 敲一下鼓，“咚”能不能马上出现、及时停住，这种起落的干脆程度就是瞬态。

## 先听懂，再记名字

拍一下桌面，声音“哒”地出现，很快结束；拍一下厚枕头，声音更闷，尾巴也不一样。这个类比能帮助你注意声音的 **开头和结尾**。

录音先决定一个声音本来怎样出现、怎样结束。听的时候，可以把这段变化拆成两部分：

- **起音（attack）**：声音刚出现时有多快、多清楚；
- **衰减（decay）**：声音出现后怎样变小、多久结束。

说耳机的**瞬态响应**，是在问它能否跟上录音里的快速变化，有没有把本来分开的开头和尾巴变得拖沓或模糊。听鼓点、拨弦和钢琴触键时，这种差别最容易分辨。

## 听起来是什么区别

| 你听到的现象 | 常见描述 |
|--------------|----------|
| 鼓点边缘清楚，相邻两下之间有空隙 | 起落利落、瞬态清楚 |
| 鼓点变厚，尾音盖到下一下 | 拖、糊、收不住 |
| 开头很突出，但后面的琴腔余韵仍完整 | 起音清楚，不等于尾音被砍掉 |

“快”不等于一定正确，“慢”也不等于一定难听。声音本来就有长短：三角铁应该有余韵，手掌拍桌子本来就该很短。真正要判断的是 **该停的时候能不能停，该留下的余韵有没有留下**。

## 别把它全归到单元类型

你可能见过“动铁一定快、动圈一定慢”之类说法。单元结构会影响设计条件，但不能直接替一副成品耳机下结论。振膜、负责推动它的磁铁与线圈、腔体，以及控制多余振动的材料和结构都会参与结果。

听感还会被这些因素影响：

- 录音本身给鼓加了多少混响，也就是空间回声和余音；
- 低频量感是否太多，盖住相邻声音；
- 高频是否突出，让起音显得更锋利；
- 播放音量是否一致。

因此，频响不同也可能让两副耳机听起来“快慢不同”。不要只看材料名称或一张宣传图。

## 这样试听最容易听出来

1. 选一首你熟悉、鼓点连续而清楚的歌，固定 10-20 秒片段。
2. 两副耳机调到接近的响度；更响的那副很容易被误判为更清楚。
3. 先听每一下鼓的开头，再听它结束后与下一下之间有没有干净空隙。
4. 来回听两三轮并记录“起音、尾音、相邻声音”三项，不凭一次惊艳下结论。

你最后可以这样说：

> “这副耳机的鼓点起音很清楚，尾巴没有盖住下一下；快歌里各个鼓点比较容易分开。”

这比只说“瞬态好”更具体，也更容易让别人听懂。
