# 声场（词典条）

## 一句话

声场 = 闭上眼睛，你面前有多大的舞台。

## 三个维度

- **宽** = 最左和最右的乐器离多远
- **深** = 你能分得清鼓在后面、歌手在前面吗
- **高** = 有些声音是"从上面来"的吗（这个最难，大部分录音没有）

## 类比

声场窄 = 所有乐器挤在你脑门正中间，像一个小房间听歌。

声场宽 = 吉他手和鼓手之间像隔了五米，像一个空体育馆听歌。
