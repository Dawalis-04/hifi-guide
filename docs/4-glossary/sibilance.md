# 齿音：人声里过分抢耳的“嘶”“吃”“诗”

> **先记一句人话：** 歌手发出 s、sh、x、ch 一类辅音时，如果那一下特别尖、像喷气声贴着耳朵划过去，通常会被称为齿音突出。

## 先分清“存在”和“刺耳”

正常说话本来就有这些辅音。把它们全部磨掉，人声会含糊；把它们放得太突出，又会让人想调低音量或摘下耳机。

所以“能听见齿音”不一定有问题，真正需要注意的是：它是否明显压过相邻的字、每次出现都让你不舒服。

## 它可能从哪里来

齿音通常不是一个设备单独造成的，常见因素有：

- 歌手本身的发音方式；
- 话筒、距离、录音和后期处理；
- 耳机在相关高频区域的能量；
- 入耳式耳机的佩戴深度和耳塞套；
- 你的耳道形状、听音音量和个人敏感度。

它常与大约 5-10kHz 的区域有关，但没有一个对所有人都适用的固定频率。尤其不要看见测量图上 8kHz 的尖峰，就直接断定一定有齿音。

## 四步判断是谁的问题

1. **先降一点音量。** 音量过高会让刺激感快速增加，也更伤听力。
2. **换两三首录音。** 只有一首明显，问题更可能在录音；很多歌都明显，再看耳机和佩戴。
3. **重新佩戴。** 入耳式可尝试合适尺寸的耳塞套和略微不同的深度，但不要勉强塞得过深。
4. **换另一副熟悉的耳机复听。** 保持响度接近，确认差异能否重复出现。

最后可以这样描述：

> “这副耳机在几首女声里都会把‘诗、是、吃’的开头推得很靠前，我降低音量后仍觉得刺激。”

这比一句“齿音重”更容易复现，也更方便别人理解。
