# 建立你的参考曲目

## 为什么你需要"参考曲目"

参考曲目不是"音质标准答案"，而是你熟悉的一把尺子。拿新耳机听一首从没听过的歌时，录音和耳机同时在变，很难知道差异来自哪里。

固定同一个平台、同一个发行版本和同一个片段，再只换耳机，结论才更可靠。不同重制版、现场版，以及播放平台自动调节音量的功能，都可能改变声音。

就像你每天都走同一条路回家。有一天你换了一双鞋，你立刻知道这双鞋和上一双差在哪——因为路没变。

---

## 开始前先做四件事

1. 关闭均衡器（EQ，用来调整不同频段音量）、低音增强和空间音效，确保两边设置相同。
2. 每首只选 15-30 秒，记住所用平台和版本。
3. 把两副耳机调到差不多响，最好以中央人声为参照。更响的一副通常会显得更清楚、更有力。
4. 每次只听一个问题，例如先听低频，不要一轮里同时判断所有项目。

下面 5 首只是容易找到的起点。你可以换成自己更熟悉的歌，只要它确实包含要听的声音。

### 1. 听低频节奏和收放

**推荐：Daft Punk - Get Lucky**

**听什么：**
- 选同一段鼓和贝斯同时出现的位置，两条节奏能不能分别跟住
- 前一个鼓点的尾音会不会盖到后一个鼓点
- 低频相对 Pharrell 的人声是偏多、适中还是偏少

这首更适合听中低频节奏，不足以单独证明 20Hz 下潜。想听更低的电子低音，可换 **Billie Eilish - bad guy** 或 **deadmau5 - Strobe**，但不要靠加大音量追求"震胸口"。

---

### 2. 测人声位置与齿音

**推荐：梁静茹 - 勇气**

**听什么：**
- 人声相对伴奏是靠前、适中还是靠后，不必硬猜相隔几米
- s、sh、ch 一类辅音是否突然比其他字更抢耳，这种尖锐感常被叫作**齿音**
- 完整嗓音是否清楚，还是只有呼吸和唇齿音特别突出；后者也可能只是高频偏多

**替代：** Adele - Someone Like You（听人声厚薄），或陈奕迅 - 好久不见（听人声与伴奏的相对位置）。同一首录音本来就可能有齿音，所以必须换耳机重复比较，不能把一次听感全归因于设备。

---

### 3. 听定位和移动是否连贯

**推荐：Yosi Horikawa - Bubbles**

**听什么：**
- 小球移动时，位置是连续变化，还是会突然跳一下
- 同时出现多个声音时，能不能分别跟住它们
- 中央、最左和最右的位置是否稳定；用"脑内、到耳边、越过耳边"描述即可

这首歌的人工摆位很明显，适合检查**定位**，但不能单独代表自然舞台大小。再加一首你熟悉的现场录音，或只有几件乐器的小型演奏来听前后层次；不要随便换一版陌生交响乐，因为不同录音的空间差异很大。

---

### 4. 听高频亮度和密集段落

**推荐：Steely Dan - Aja**

**听什么：**
- 镲片是有金属光泽和自然尾音，还是只剩尖锐的"嘶"声
- 后段密集鼓声里，不同鼓和镲片能不能分别跟住；这会同时受录音、高低频多少和音量影响
- 正常音量听同一段一分钟后是否想调小。若觉得累，先降音量、换一首录音复核，再判断是不是耳机高频刺激

**替代：** Eagles - Hotel California（Hell Freezes Over 现场版，听拨弦尾音），或 Diana Krall - The Look of Love（听人声与器乐的平衡）。这里的**泛音**，就是跟在主音上方、让不同乐器保留各自质感的高频成分。

---

### 5. 测你自己的直觉

**推荐：一首你听了 100 遍的歌。**

没有统一的"对的声音"，但可以有稳定的个人偏好。

换上一副新耳机，先写下第一反应，再按后面的 A/B 方法复听。第一印象能告诉你哪里值得注意，不能单独证明差异，也不该直接决定购买。

越熟悉的曲目越容易发现变化；重复、音量匹配和尽量不知道当前是哪副耳机，才能减少心理预期带来的误判。

---

## 怎么用这 5 首歌做 A/B 测试

A/B 就是在两副耳机之间来回切换，目标是尽量只改变耳机这一项。反复摘戴也会改变密封和位置，入耳式耳机尤其明显。

1. 用同一设备、同一平台、同一版本，固定一个 15-30 秒片段。
2. 固定耳塞套和佩戴位置。正式比较前，把同一副耳机重戴两次，先确认佩戴本身会带来多大变化；每次都检查密封。
3. 先匹配音量。没有测量设备时，用中央人声作参照，来回调整到差不多响。
4. 第一轮按 A→B，第二轮按 B→A；至少做三轮，每轮只记录一个问题。
5. 有人帮忙时，让对方随机递给你 A 或 B，不告诉你是哪副。多轮都能分辨，或稳定写出同一差异，比看到品牌后的第一印象可靠。
6. 每 10-15 分钟休息一下。差异忽有忽无时，诚实记录"暂时听不出"，不要强迫自己找答案。

可以用同一张小表记录：

| 项目 | A | B | 三轮都一致吗 |
|------|---|---|--------------|
| 低频多少与收放 |  |  |  |
| 人声相对位置 |  |  |  |
| 高频是否刺激 |  |  |  |
| 定位是否稳定 |  |  |  |
| 更愿意久听哪副 |  |  |  |
