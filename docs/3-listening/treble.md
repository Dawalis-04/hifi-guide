# 高频：镲片亮不亮

## 先做一个实验

拿你的指甲：弹一下玻璃杯的杯沿。再弹一下塑料杯的杯沿。

玻璃杯的声音："叮"——锐、亮、有余韵。塑料杯的声音："嘟"——闷、软、弹完就没了。

**这就是高频的区别。玻璃杯 = 高频亮。塑料杯 = 高频暗。**

## 高频三个层次

**亮（8kHz 有峰）：** 镲片的声音像在你眼前晃闪亮的东西。吉他拨弦的每一个"叮"都还给你。

**有空气感（12kHz 以上在走）：** 你能"感觉"到录音棚的大小——不是因为回声，是因为极高频的泛音给你一种"空间信息"。

**刺耳（8-10kHz 太高了）：** 你用手指甲刮黑板——就是那种感觉。听久了想摘耳机。

## 练习

用你现在的耳机听**Steely Dan - Aja**开场那段鼓：

1. Steve Gadd 的镲片——是"亮"、"暗"、还是"刺"？
2. 听完一整首，耳朵累不累？累了 = 这副耳机的高频对你来说太刺激了
