# 高频：镲片亮不亮

## 先说人话

高频让镲片有亮光、让辅音有边缘，也会影响声音是否刺耳、有没有"空气感"。判断高频时，不要只盯着一个数字，而要分开听：**亮不亮、刺不刺、尾音是否自然。**

用指甲轻弹玻璃杯和塑料杯，确实能帮助记住"亮"和"暗"。但材料、共振和余音都同时变了，所以这只是记忆类比，不是耳机高频的测试。

## 高频大致分在哪

可以先把 4kHz 以上叫作高频。下面的范围只是帮助入门，不是固定开关：

- **4-6kHz：清晰和边缘。** 偏多时，鼓点和辅音更突出；过多可能显得硬或冲。
- **6-10kHz：齿音和镲片亮度的重要区域。** 偏多可能更闪、更刺激，但耳朵形状、佩戴位置和录音都会改变结果。
- **10kHz 以上：常被描述为空气感。** 它会影响高频尾音是否舒展，但不能单独告诉你录音棚有多大，也不等于解析力。

所谓**泛音**，就是跟在一个主音上方、让小提琴和吉他听起来不同的更高频成分。所谓**齿音**，是"s、sh、ch"一类辅音显得格外尖锐。

## 亮、刺耳和空气感不是一回事

- **亮：** 镲片、拨弦和辅音更显眼，可能好听，也可能只是注意力被高频吸走
- **刺耳：** 某些声音让你下意识皱眉或想调小音量；上中频、高频、录音和音量都可能是原因
- **空气感：** 高频尾音显得舒展、不闷；每个人对极高频的听力不同，年龄和听力状态也会影响感受

表示高低声音强弱的频响图，在 8kHz 以上尤其容易受测量和佩戴影响。因此不能看到一个尖峰就断言耳机一定刺，最终还要用实际佩戴和熟悉的录音确认。

## 一句话怎么描述

可以说：

> "镲片偏亮，尾音能自然散开；人声的 s 音偶尔突出，但正常音量下不会让我想摘耳机。"

## 跟着做一次试听

选一首有镲片和清楚人声、而且你熟悉的歌，例如 **Steely Dan - Aja**。固定同一个版本和 20-30 秒片段，关闭均衡器（EQ）和空间音效，从偏小的音量开始：

1. 听镲片出现时，是有金属光泽，还是只剩尖锐的"嘶"声。
2. 听镲片消失前的尾音，是自然变小，还是突然被截断或一直糊成一片。
3. 再听一段熟悉的人声，记下 s、sh、ch 是否比其他字明显抢耳。
4. 比较两副耳机时先把人声调到差不多响；先听 A 再听 B、再反过来听，至少做三轮。更响的一副通常也会显得更亮。

听一会儿觉得累，不一定全怪高频，也可能是音量过大、夹得不舒服或录音本身尖锐。先调低音量，再换一首录音复核；如果耳朵疼、发闷或出现耳鸣，应立即停止。
