# 声场：闭上眼睛能看见舞台吗

## 先说人话

声场是闭上眼睛后，你感觉音乐占了多大空间。与它经常一起出现的另一个词是**定位**：你能不能指出歌手和乐器大概在哪里。

**舞台大不等于一定更好。** 有些耳机把声音铺得很开，却把位置弄得模糊；有些舞台不大，但歌手始终稳在中央、每件乐器都容易分辨。空间大小和定位清楚度要分开评价。

## 先分清谁在起作用

歌曲制作先决定乐器被放在左边、右边、前面还是后面。耳机高低频的多少、左右声道配合、佩戴位置，以及每个人耳朵和头部的形状，再决定这些线索到你这里听成什么样。

所以，同一首歌必须用同一个版本比较。某件乐器挤在中央，也可能是录音本来就这样，不能只听一副耳机就归因于设备。

## 声场的三个方向

- **宽度（左右）：** 最左和最右的声音分得有多开。用"都在脑内、到耳边、似乎越过耳边"描述就够了，不必硬猜几米。
- **深度（前后）：** 歌手、鼓和其他乐器能否听出前后层次，还是像排在一条线上。
- **高度（上下）：** 某些双耳录音或空间音频会提供上下线索；普通立体声里的高度通常较弱，而且不同人听到的差异很大。

还要单独记一项**定位稳定性**：声音移动时是否连贯，中央人声会不会散开或偏向一边。

## 一句话怎么描述

可以说：

> "舞台宽度大约到耳边，中央人声很稳；左右位置清楚，但前后层次不太明显。"

这比只说"声场大"多交代了空间大小、定位和深度。

## 跟着做一次试听

先关闭空间音效和均衡器（EQ，用来调节各频段音量），固定同一个版本、同一个音量。若比较两副耳机，先把中央人声调到差不多响；先听 A 再听 B、再反过来听，至少做三轮。

1. 听 **Yosi Horikawa - Bubbles** 的同一段，跟随小球移动：路径是连续的，还是中间会突然跳位置？这首人工摆位明显的录音适合检查定位，不适合单独证明舞台是否自然。
2. 再选一首你熟悉的现场录音，或只有几件乐器的小型演奏，先找中央人声，再找最左、最右的乐器，最后判断是否有前后层次。
3. 每轮只写四项："宽度、深度、中央是否稳、移动是否连续"。不要先看品牌和价格。
4. 如果左右明显不平衡，先重新佩戴，再换一首确定人声居中的录音，并交换左右声道或换另一副耳机复核。若偏向始终跟着同一只耳朵，参照 [1.3 认识你的耳朵](/hifi-guide/docs/1-basics/your-ears.html) 的听力提醒；不要立刻把它叫作声场问题。
