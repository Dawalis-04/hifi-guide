# 声场：闭上眼睛能看见舞台吗

## 声场不是"声音绕圈"

有人跟你说"声场很大"，他的意思不是声音在你脑袋周围转圈。他的意思是：闭上眼睛，你面前有一个"舞台"。乐器分布在这个舞台上——吉他手在左边、贝斯手在右边、主唱在正中间、鼓在最后面。声场越好，这个舞台越大、乐器位置越清楚。

## 声场有三个维度

**宽度（左右）：** 最左边和最右边的乐器离你耳朵有多远。如果感觉所有乐器都被压在你脑袋正中间——声场窄。如果感觉吉他手和键盘手之间隔了五米——声场宽。

**深度（前后）：** 你能分得清主唱在前面、鼓在后面排吗。好的声场有"层"，不是所有乐器排成一条直线。

**高度（上下）：** 这个维度最难还原。好的录音能让某些声音"从上面来"——管弦乐的铜管、合唱团的最高声部。但大部分流行乐录音没有高度信息，所以听不出来不是你的锅。

## 练习

用你现在的耳机听**Yosi Horikawa - Bubbles**：

1. 每个弹跳的小球，你能"看见"它在什么位置吗？
2. 最左边和最右边的小球，感觉离你耳朵有多远？
3. 闭上眼睛，你能不能"画出"这些球的位置？
