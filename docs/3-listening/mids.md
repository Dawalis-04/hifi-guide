# 中频：唱歌的人离你多远

## 先说人话

中频是人声和大多数乐器最主要的活动范围。听中频，先问三件事：**人声够不够厚、歌词清不清楚、歌手有没有被伴奏盖住。**

你能听出家人在厨房还是客厅，是左右声差、房间反射和音色等线索一起工作的结果。耳机里的人声远近也一样，不能只归因于中频。

## 中频大致管什么

可以先把中频粗略理解成 250Hz-4kHz，边界会因资料而略有不同：

- **250-500Hz：厚度。** 偏多时，人声和吉他更厚暖；过多可能发闷。
- **500Hz-2kHz：主体。** 歌词、钢琴、吉他的主要轮廓大多在这里。
- **2-4kHz：存在感。** 相对偏多时，人声常显得更靠前、更清楚；过多也可能紧、冲或容易累。

这些范围彼此重叠，不是三个开关。低频太多会盖住人声，高频偏多会让唇齿音更明显，制作歌曲时把歌手调得多响也会改变距离感。

## 怎么描述人声位置

- **靠前：** 人声比伴奏更抓注意力，像站在乐队前面
- **适中：** 人声和伴奏都清楚，谁也没有一直压过谁
- **靠后：** 乐器先吸引你，人声像融在伴奏里

这里说的是相对位置，不是能实际量出的距离。靠前也不等于解析力高，靠后也不等于质量差；它首先是录音与调音共同造成的听感，其次才是你的偏好。

## 一句话怎么描述

可以说：

> "人声比伴奏稍靠前，厚度够，歌词清楚；齿音没有被特别放大，听完整首也不累。"

这比"中频毒"或"人声两米远"更容易让别人理解。

## 跟着做一次试听

选一首你熟悉、录音里人声清楚的歌，例如 **陈奕迅 - 好久不见**。固定同一个版本和 20-30 秒片段，关闭均衡器（EQ，用来调整不同频段音量）和空间音效，再用舒服的音量听：

1. 先只记"厚、刚好、薄"，不要急着评价好坏。
2. 再听人声与吉他的关系：歌词是否要费力找，还是人声一直压过伴奏？
3. 注意清楚的是完整嗓音，还是只有呼吸和唇齿音特别突出；后者也可能只是高频偏多。
4. 比较两副耳机时，先把人声调到差不多响；先听 A 再听 B、再反过来听，至少做三轮。只有重复出现的差异才记录。

如果只有一副耳机，也可以第二天用同一音量和同一片段复听。不要用记忆比较不同发行版本，因为歌曲版本的差异可能比耳机还大。
