# 看懂耳机参数

## 阻抗（Ω）

标多少欧姆，决定了这副耳机"好不好推"。

- **16-32Ω**：手机直推无压力。大部分消费级耳机在这
- **32-100Ω**：手机能推得响，但推不好。加个小尾巴有明显提升
- **100-300Ω**：需要用专门的耳放。手机直推声音会很"虚"
- **300Ω+**：台机用户专区，不是给你准备的

**简单记：阻抗 < 32Ω = 随便插。32-100 = 最好配个小尾巴。> 100 = 要花钱。**

## 灵敏度（dB/mW）

标多少分贝，决定了"给同样的电，这副耳机有多响"。

- **< 95dB**：偏难推。需要更多功率
- **95-105dB**：正常水平
- **> 105dB**：高灵敏度，容易推响——但"响"不等于"好听"

**阻抗和灵敏度要一起看。低阻抗+高灵敏度=手机直推最友好。高阻抗+低灵敏度=前端预算要翻倍。**

## 频响范围

标"5Hz-40kHz"不代表什么——人根本听不到 5Hz，也听不到 40kHz。

看这个数字的唯一用处：如果标称的低频最低只能到 30Hz，那低频下潜确实偏浅。如果标 5Hz，那只是厂家自信——不代表你的耳朵能验证。

**真正有用的是频响曲线（见第三章第五节），不是这个数字。**
