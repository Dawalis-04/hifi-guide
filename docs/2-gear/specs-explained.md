# 看懂耳机参数

## 先记一句

**“好不好推”不能只看阻抗。要把阻抗、灵敏度及其单位放在一起，再看播放设备能提供多少电压和电流。参数能判断兼容性，但不能直接判断音质。**

## 阻抗（Ω，欧姆）是什么

阻抗可以先理解为耳机向播放设备“要电”的方式之一。它会影响需要多少电压和电流，但不能单独告诉你这副耳机有多难推。

- 较高阻抗的耳机通常需要更多电压，不代表一定难听或必须配台式耳放。
- 在其他条件相同时，较低阻抗的耳机可能需要更多电流；是否听得到底噪，主要还要看耳机灵敏度和播放设备自身的噪声。
- 部分耳机的阻抗会随频率变化。播放设备输出阻抗过高时，甚至可能改变耳机的频率响应。

因此，“32Ω 以下随便推，100Ω 以上必须买耳放”是不可靠的判断。

## 灵敏度必须连单位一起看

灵敏度表示输入一定电信号时，耳机能产生多大声压。常见标法包括：

- **dB SPL/mW**：输入 1 毫瓦功率时有多响。
- **dB SPL/V**：输入 1 伏电压时有多响。

其中 SPL 是“声压级”，也就是测得有多响。两个数字即使都写 100dB，只要单位不同，就不能直接比较。还要留意有的厂商使用 dB/mV、不同测试频率或不同的仿真人耳测量夹具，数据未必能完全横向对照。

## 怎么判断是否需要耳放

1. 先确认厂商给出的阻抗、灵敏度和灵敏度单位，缺少单位的数据参考价值很低。
2. 接到现有设备后，从低音量开始。如果正常歌曲已有足够音量，响度高峰不破音，且仍留有调节余量，通常不需要为了“推得更好”立即买耳放。
3. 高灵敏度入耳式还要听暂停时有没有底噪，以及低音量时左右声道是否平衡。
4. 若能查到输出阻抗，可把“播放设备输出阻抗低于耳机最低阻抗的约八分之一”作为保守兼容参考，尤其是多动铁入耳式。

## 频响范围怎么看

“5Hz-40kHz”只写了两个端点，却没说明在端点衰减多少，通常不能告诉你低频是否充足、高频是否刺耳。即使标“20Hz-20kHz”，中间也可能有很大的起伏。

带测试条件和容差的范围更有意义，但判断音色仍应看同一测量体系下的频响曲线，并结合实际佩戴试听。

## 别误会

- 低阻抗不等于容易驱动，高阻抗也不等于音质更高级。
- 能推得很响只说明音量放大可能足够，不代表失真、噪声和声道平衡一定合格。
- 更大功率不会自动增加解析力；功率过大和音量过高反而可能损伤耳机或听力。
- 参数表无法代替试听，舒适度和调音通常比纸面数字更影响日常体验。
