# 前端是什么

## 先记一句

**前端是耳机之前的播放链路，通常包括播放器、数字处理、DAC（数模转换）和耳放。手机、小尾巴、蓝牙耳机内部都可能把这些功能集成在一起。**

前端首先要解决“能不能正常、干净地驱动”，而不是按耳机价格搭配固定比例。几百元耳机可能有特殊驱动需求，昂贵耳机也可能用普通小尾巴就足够。

## 各设备在做什么

| 名称 | 翻译成人话 | 什么时候有用 |
|------|------------|--------------|
| 外置 DAC | 把数字音频变成模拟电信号；纯 DAC 后面仍要接耳放 | 已有耳放或有源音箱，需要增加数字输入时 |
| 耳放 | 把模拟信号放大到耳机需要的电压和电流 | 音量不足、响度高峰破音，或设备无法稳定驱动负载时 |
| 小尾巴 | 常把 DAC 和耳放装进 USB 转接头 | 需要耳机口、便携和基础驱动力时 |
| DAP | 自带系统、存储、DAC 和耳放的便携播放器 | 想脱离手机管理本地曲库、续航和实体按键时 |
| 台式 DAC/耳放 | 接口、功率和操作空间更多 | 固定桌面使用、需要多输入输出或确有驱动需求时 |

## 先检查现有设备

满足下面这些条件，就没有必须升级的技术理由：

- 常听音量下还有适当余量，音乐高峰不破音。
- 暂停播放时没有明显底噪或电流声。
- 低音量时左右声道平衡，连接稳定。
- 接口、麦克风、系统兼容和便携性都满足需求。

## 怎么选择

1. **先解决明确问题。** 手机没接口就买兼容的小尾巴；高灵敏度耳塞有底噪，就找低噪声输出；难驱动耳机音量不足，再补足电压或功率。
2. **再选功能。** 根据是否需要实体音量旋钮、本地存储、多设备输入、麦克风支持和便携性决定形态。
3. **比较时匹配音量。** 两台设备只差一点响度，也容易被听成“声场更大、低频更稳”。盲测或快速切换会更可靠。

## 别误会

- 不存在通用的“耳机和前端 1:1 预算比例”。达到干净、足量的输出后，继续加钱通常会快速进入收益递减。
- “平衡接口”常用于获得不同的电路连接或更高输出能力，但不天然比常见的 3.5mm 单端接口音质好。接口和转接方式必须遵守设备说明，不能只看插头能否插入。
- DAP 和台机提供的是功能与驱动选择，不会仅因体积或价格就让所有耳机升级一个等级。
- 普通升级线主要改变耐用、长度、柔软度和佩戴体验。除非阻抗等电气条件明显变化，否则不应把它当成可靠的调音工具。

把预算留给更合适的耳机、耳塞套、舒适度或有依据的均衡调整，往往比盲目堆前端更有效。
