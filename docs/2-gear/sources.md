# 前端是什么

## 一句话

前端 = 所有在"耳机"之前的东西。手机、小尾巴、播放器、耳放、解码器——全是前端。

## 我真的需要多花钱吗

| 你的耳机 | 你的前端 | 建议 |
|---------|---------|------|
| 几百块消费级 | 手机直推 | 够了。换个好耳机比换前端提升大得多 |
| 1000-2000 HiFi入门 | 手机+小尾巴(200-500块) | 值得。小尾巴是性价比最高的前端升级 |
| 2000-5000 | 入门播放器或台式 | 前端和耳机预算 1:1 差不多 |
| 5000+ | 专用系统 | 但这时候你应该不需要看这篇了 |

**核心原则：耳机的优先级永远高于前端。2000 的耳机+手机直推，打不过 1000 的耳机+1000 的前端——但 1000 的耳机+1000 的前端，打不过 1500 的耳机+500 的前端。耳机的天花板决定一切。**

## 三个你不需要现在买的东西

- **DAP**（便携播放器）：等你有一副你决定"不退烧了"的耳机再考虑
- **台机**：等你耳机插手机真的推不动了再买
- **升级线**：等你把频响图看懂了，知道自己想调哪个频段了再烧
