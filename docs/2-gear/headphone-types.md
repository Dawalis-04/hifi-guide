# 耳机类型速览

## 一张表搞懂

| 类型 | 工作原理 | 一句话 | 代表 |
|------|---------|--------|------|
| **动圈** | 磁铁+线圈推动振膜，和你的音箱一样 | 最自然、最耐听、最不需要折腾 | HD600, MDR-7506, 瞬 |
| **动铁** | 衔铁驱动振膜，体积可以做到很小 | 解析高、响应快、但声场通常小一点 | ER4, SE846 |
| **圈铁** | 动圈管低频+动铁管中高频，各干各的 | 低频有圈味、细节有铁味——但如果分频没做好会割裂 | 隼u, KATO |
| **平板** | 大面积薄膜在磁场中振动 | 低频又深又干净、瞬态极快，但很重、很吃推力 | Sundara, Edition XS |
| **静电** | 极薄振膜在静电场中振动 | 细节量巨大、瞬态变态快，但需要专用耳放、贵、娇气 | STAX, Shangri-La |

---

## 所以该怎么选

| 如果你 | 试试 |
|--------|------|
| 刚开始玩，什么都不知道 | **动圈**。便宜、好听、没毛病 |
| 喜欢"哇这个细节我从来没听到过" | **动铁** 或 **圈铁** |
| 听器乐/古典为主，想要声场大 | **平板**——但先确认你的前端推得动 |
| 预算五位数以上、家里有台机 | **静电**——这不是入门品类，但你知道它存在就行 |

---

## 一个重要的提醒

**单元类型决定下限，调音决定你喜不喜欢。**

不是所有动圈都比动铁好听。不是所有平板都比动圈贵。

一副调音拉胯的平板耳机，可能比一副调音好的动圈难听得多的多。所以先看调音（第三章），再看单元类型。不要反过来。
