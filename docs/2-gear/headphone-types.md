# 耳机类型速览

## 先记一句

**单元类型只说明“怎样让发声薄片（振膜）振动”，不能单独决定音质、声场或解析力。具体结构、调音、佩戴密封和生产一致性（品控），往往比类型名称更重要。**

## 常见单元怎么工作

| 类型 | 翻译成人话 | 选购时留意 |
|------|------------|------------|
| **动圈** | 线圈在磁场里推动振膜，耳塞和头戴耳机都很常见 | 产品跨度最大，声音没有统一模板 |
| **动铁** | 很小的金属片（衔铁）带动发声薄片，容易放进狭小的入耳式腔体 | 常见单单元或多单元设计，最终效果取决于声音经过的细管（声学管道）与调音 |
| **圈铁/混合单元** | 把动圈、动铁等不同单元放在一起分工 | 单元更多不等于更好，不同单元交接不好时声音可能不连贯 |
| **平板振膜** | 导电线路分布在薄膜上，由磁场驱动 | 有些型号较重或灵敏度较低，但不能一概说都难推 |
| **静电** | 带电薄膜在高压静电场中运动 | 通常需要匹配的专用驱动器，不能直接接普通耳机口 |

## 先选形态，再看单元

- **通勤**：先看隔音、重量、麦克风和是否容易戴稳。
- **在家安静听**：开放式通常更透气，但会漏音，也几乎不隔绝环境声。
- **办公室或宿舍**：封闭式、入耳式更少打扰别人，但密封和舒适度要亲自确认。
- **容易耳道不适**：优先试头戴式；容易夹头或怕热，则试重量更轻的入耳式。

## 怎么判断

1. 先定使用场景和形态，再按预算挑两三款调音合适的具体型号，不要先限定单元类型。
2. 入耳式要尝试不同尺寸耳塞套。密封不足会让低频明显减少，也会改变整体音色。
3. 用相近音量试听熟悉的人声、低频和复杂编曲，至少戴 15 分钟确认舒适度。
4. 最后再检查重量、阻抗、灵敏度以及是否需要专用驱动设备。

## 别误会

- 动铁不天然“解析更高”，动圈也不天然“更自然”。这些是具体产品的表现，不是工作原理的保证。
- 平板不一定低频更多、声场更大，也不一定必须配昂贵耳放。
- 多单元只是设计手段，不是音质等级；一个做好了的单单元可以胜过衔接不佳的多单元。
- 单元尺寸也不能直接比较音质。入耳式、头戴式和音箱面对的声学条件不同。
