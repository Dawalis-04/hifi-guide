# 无线 vs 有线

## 不是谁好谁坏

这是"方便"和"音质"的取舍，不是"聪明"和"傻子"的选择。

| | 有线 | 无线 |
|---|------|------|
| 音质上限 | 理论无压缩 | 有损传输（LDAC 接近无损但仍有差距） |
| 方便程度 | 被线拴着 | 走到哪听到哪 |
| 需要充电吗 | 不用 | 耳机盒都得充 |
| 保值 | 好的有线能用十年 | 三年后电池衰减 |
| 适合场景 | 在家认真听 | 通勤、运动、打电话 |

## 一个不会错的策略

**买一副好的有线留在家，买一副够用的无线出门。**

有线负责让你知道"这首歌本来的样子"。无线负责让你在通勤时不无聊。两副耳机各干各的，比买一副"全能"的省钱，也比纠结"到底买哪个"省心。
