# 无线 vs 有线

## 先记一句

**无线和有线首先是使用方式的差别。调音、佩戴和隔音通常比连接方式更影响听感；蓝牙编码只是整条播放链中的一环。**

## 实际差别

| | 有线 | 无线 |
|---|------|------|
| 连接 | 模拟耳机口或 USB 数字连接，通常稳定、延迟低 | 蓝牙连接，距离、干扰和系统会影响稳定性与延迟 |
| 音频处理 | 取决于手机、电脑、小尾巴和播放器 | 耳机内部会解码，并通过 DSP（软件处理声音）调音和放大 |
| 功能 | 通常简单，不一定有麦克风和控制 | 常见主动降噪、通透、通话、多设备和触控 |
| 维护 | 不用给普通有线耳机充电，线材有时可更换 | 电池会老化，长期寿命还取决于软件和维修能力 |
| 场景 | 录音、游戏、固定座位和低延迟需求 | 通勤、运动、通话和需要降噪的环境 |

## 蓝牙编码到底是什么

手机会先把音频编码，再通过蓝牙传给耳机。SBC、AAC、aptX、LDAC 等编码每秒传输的数据量（码率）、延迟和设备支持不同，但名称本身不保证最终音质。

**LDAC 本身是有损编码，最高标称码率也不能保证传过去的数据和原文件一位不差（逐位无损），不能直接叫作“无损”或“接近无损”。** 它还可能为了连接稳定自动降低码率。是否能用取决于手机和耳机双方支持、系统设置、无线环境及耳机内部实现。

即使编码存在差异，在地铁等嘈杂环境中，隔音和主动降噪带来的改善往往更容易听见。反过来，录音监听、乐器演奏和竞技游戏更在意稳定的低延迟，有线通常更省事。

## 怎么判断

1. **通勤和飞机**：先试降噪、通透模式、佩戴稳定和风噪，再考虑编码名称。
2. **游戏、录音和视频制作**：确认实测延迟；蓝牙“游戏模式”能降低延迟，但不一定达到有线水平。
3. **在家听歌**：两种方式都可以。用同一首歌、接近音量和同一 EQ（均衡器，用来调节各频段音量）比较，不要默认有线必胜。
4. **买无线前**：确认手机与耳机共同支持的编码、应用功能、单次续航和电池维修政策。

同一副主动降噪耳机切到被动有线模式时，内部 DSP 可能关闭，调音也会改变。因此这种比较不一定只比较了“有线和蓝牙”。

## 别误会

- 更高码率不等于整机一定更好听，驱动单元、调音和 DSP 仍然重要。
- 有线链路也可能有底噪、功率不足或不合理的音效处理，不是天然“原音直出”。
- 无线不是只能“听个响”。设计良好的无线耳机可以很准确，主动降噪还可能让你用更安全的音量听清内容。
- 不必固定买两副。按最常用场景选一副真正合适的，通常比为了连接方式凑组合更省钱。
