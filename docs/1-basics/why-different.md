# 为什么手机外放和耳机差那么多

## 先记一句

**差异不只来自单元大小，还来自机身空间、功率、声学设计、DSP（用软件修正声音）、播放距离、房间反射和环境噪声。耳机离耳朵更近，不必向整个房间辐射声音，还能把左右声道分别送到两只耳朵。**

## 差别从哪来

### 1. 低频需要移动空气

手机扬声器和内部腔体都很小，振膜能移动的距离也有限。厂商可以用腔体、多个扬声器和 DSP 补偿，但强行增加低频会很快碰到失真、功耗和机身振动的限制。

耳机并不一定拥有更大的单元，入耳式单元甚至也很小。所有耳机的共同优势是离耳朵近，不需要把整个房间的空气都推响；其中入耳式和不少封闭式耳机还能形成较小、相对封闭的声学空间。因此它们更容易播放较深的低频，但能不能做好仍取决于密封和调音。开放式耳机不靠密闭工作，也会有不同的低频取舍。

### 2. 房间会参与外放

手机外放会受到附近桌面、墙壁和整个房间的影响。反射会改变音色，坐姿和手机摆放也会影响结果。耳机绕过了大部分房间声学问题，但耳罩、耳塞密封和佩戴位置会成为新的变量。

### 3. 左右声道的送达方式不同

手机两只扬声器距离很近，左声道也会进入右耳，立体声分离容易变窄。耳机把两个声道分别送到左右耳，定位线索通常更明显。不过“分得开”不等于声场一定自然，耳机仍可能让声音集中在头内。

### 4. 环境噪声会遮住细节

空调、说话声和车流声会盖住音乐里较轻的部分，这种现象叫**掩蔽**。入耳式的密封或主动降噪可以减少掩蔽，让你在更低音量下听清内容。这里听到的“更多细节”，有时只是环境安静了，并不是耳机凭空创造了信息。

## 怎么判断

1. 用同一台手机、同一首歌，并关闭额外的均衡器和空间音效。
2. 在安静房间把两种播放方式调到接近的主观响度，避免把“更响”听成“更好”。
3. 听持续的贝斯音是否完整、人声是否发薄，再听左右分开放置的乐器是否容易辨认。
4. 改变手机的朝向和桌面位置，再重新佩戴耳机。变化很大，说明摆位、反射或密封正在影响结果。

## 别误会

- 单元更大只提供某些物理条件，不会自动带来更深低频或更好音质。
- 耳机也可能失真、漏气或调音失衡；离耳朵近不代表必然准确。
- 手机外放适合分享、导航和短视频，耳机适合私密聆听。两者使用目标不同，不必只按音质分高下。
