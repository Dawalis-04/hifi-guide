# 为什么手机外放和耳机差那么多

三个原因，每个三句话说完。

**1. 单元大小。** 手机外放的扬声器比你的小指甲盖还小，耳机的单元比整个耳道还大。推动空气的面积差了几十倍。面积越大的振膜，低频能"沉"得越深。手机外放永远不能让你感觉到鼓声砸在胸口——物理上就不可能。

**2. 腔体设计。** 手机扬声器背后没有"腔体"——声音往四面八方散。耳机的腔体是一个精心设计的密闭空间，控制声音怎么进去、怎么出来。腔体=耳机的"房间"，手机=广场上喊话。

**3. 耳朵独占。** 外放的声音和房间的墙壁、地板、你的身体撞在一起再进你耳朵。耳机直接把声音送到耳道里，没有"撞墙"这个步骤。所以你能听到录音棚里那台嗡嗡响的空调——那是原本就有、但外放漏掉了的。
