# 什么是 HiFi

## 先记一句

**HiFi 是 High Fidelity，中文是“高保真”。它是一种尽量少失真、少噪声地重放录音的目标，不是价格标签，也不是某一种固定音色。**

商家常把 HiFi 当产品类别使用，但写着 HiFi，不等于它一定准确、昂贵或适合你。最后仍要看具体产品和实际听感。

## 准确一点

耳机拿到的是已经完成混音和母带处理的音频信号，不是录音现场本身。**混音**是把人声和各种乐器组合起来，**母带处理**是发行前对整体音量和声音做最后整理。所谓“保真”，主要是让播放设备少添不必要的变化：

| 要看什么 | 翻译成人话 |
|----------|------------|
| 频率响应 | 低、中、高频的相对强弱，是否接近选定的调音目标 |
| 失真 | 播放时有没有多出录音里没有的杂音、毛刺或浑浊 |
| 噪声 | 安静段落里有没有底噪、电流声或蓝牙干扰声 |
| 动态余量 | 音乐突然变响时，设备会不会压扁、破音 |

任何耳机都会受到调音、佩戴位置、耳形和测量方法影响。这里的**调音**，就是设计者怎样安排低、中、高频的相对多少。它不可能把母带“原封不动”送进每个人的耳朵，工程上只能尽量接近目标，而不是保证绝对还原。

## 怎么判断

1. 用自己熟悉且录音质量正常的两三首歌，不要只听试听店准备的“发烧碟”。
2. 比较两副耳机时先把音量调到尽量一致。稍微更响的一副，常会被误以为更清楚、更有力。
3. 听人声是否自然，贝斯每个音能否分开，复杂段落里的乐器会不会糊成一团。
4. 连续戴 15 分钟，确认夹头、耳道胀痛和高频刺耳不会让你疲劳。能长期舒服地听，比一分钟的“惊艳”更重要。

## 别误会

- **HiFi 不等于完全中性。** 目标曲线是一套供设计和测量参考的频率比例；不同目标都可能有依据，耳机也常会做适度低频或高频调整。
- **细节多不等于高频越亮越好。** 过亮的调音会把齿音和边缘感放大，听起来像“解析高”，却未必更准确。
- **贵不等于保真。** 价格还包含外观、材料、品牌、功能和产量，不能直接换算成音质。
- **喜欢有染色的声音没有错。** HiFi 是技术目标，听歌满意是个人目标，两者不必互相否定。

如果你主要通勤、运动或听播客，隔音、舒适和操作可能比极限音质更重要；如果你愿意安静地听完整张专辑，再去追求更低失真和更合适的调音会更有意义。
