# 听凡 · HiFi 入门指南

> 把“低频下潜、声场、解析力、瞬态”翻译成你能听懂、能验证、能说出口的话。

这是一份写给零基础听众的耳机指南。你不需要先懂声学，也不需要购买昂贵设备；拿出自己正在用的耳机，按顺序读和听就够了。

---

## 第一次来，先记住四句话

1. **贵不等于适合。** 价格、品牌和“HiFi”标签都不能替你决定喜好。
2. **单元类型不等于声音。** 动圈、动铁、平板只是实现方式，成品还要看调音、结构和佩戴。
3. **参数能排雷，不能替你试听。** 阻抗要和灵敏度一起看，频响范围也不是音质分数。
4. **曲线负责预测，耳朵负责验证。** 先看频响猜听感，再用熟悉的歌、接近的音量来回比较。

读完后，你应该能把“好不好听”拆成更具体的问题：低频是多还是深？人声是厚还是靠前？高频是明亮还是刺激？区别能不能在复听时再次出现？

---

## 这份指南怎么把术语讲成人话

每个重要词都会按三层来讲：

- **先给一句人话**：例如“瞬态”就是声音起得快不快、收得利不利落；
- **再说明准确边界**：告诉你这个词能解释什么，不能和什么画等号；
- **最后给试听动作**：用同一段歌、相近音量和明确问题，自己验证差异。

类比是入口，不是物理定律。文章会尽量写清“通常、可能、需要结合试听”，避免把一个频率、一个参数或一种材料说成万能答案。

## 建议阅读方法

1. 从第一章按顺序读；遇到已经会的内容可以跳过。
2. 读到第三章时拿出自己的耳机，每次只听一个问题，不要同时追十个术语。
3. 比较两副耳机时，使用同一首歌的同一小段，并把响度调到尽量接近。
4. 允许自己复听和改判断。A/B（在两副耳机之间来回比较）时，能稳定重复的差异，比第一秒的“震撼”更可信。
5. 不确定术语时打开左侧或手机端章节目录，直接进入第四章词典。

## 关于听力安全

听感训练不需要大音量。觉得刺耳、耳鸣、耳闷或摘下耳机后声音发钝时，应立即停止并让耳朵休息；不适持续时应咨询专业医护人员。试听时先从低音量开始，再调到能清楚听见即可。
