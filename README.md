# 听凡 . HiFi 入门指南

> 把"低频下潜""中频密度""瞬态响应"翻译成人话。
> 不教你当发烧友，教你听懂自己的耳朵。

---

## 这是什么

一本开源的 HiFi 耳机入门手册。写给所有买耳机靠闭眼、听别人聊音质只会点头的人。

读完这本手册，你能：
- 进耳机店跟店员说出"我不喜欢低频太厚的"，而不是"有没有好一点的"
- 看懂频响曲线——其实比健康码绿不绿还简单
- 建立你自己的听音偏好，不再被测评博主牵着走

## 使用方法

1. **从第一章按顺序读**——每篇 3-5 分钟，像刷一篇公众号文章
2. **读到第三章时，拿出你现在的耳机**——跟着[参考曲目](docs/3-listening/reference-tracks.md)一首一首听
3. **遇到不认识的词，翻第四章**——每个词条都用你见过的东西做类比
4. **读完第三章第五节**，恭喜你——以后任何频响图你自己就能看了
